Gardening Know how - is redesigned web mockup for a gardening website project at WEB 2210 class. This project involves creating a mockup for the home page of the website. The mockup include a hero image which is a featured vector artwork that I drew in Adobe illustrator. The mockup also includes several user interface elements such as a navigation bar, custom icons, buttons, a container with an input box, a footer, and a section with a grid of featured articles.
